## GenAI chatbots for automatic term extraction

This repository contains data related to the following paper:

> De Pourcq, Lena, Marie Gr√©goire, and Leonardo Zilio. (to appear) Exploring the power of generative artificial intelligence for automatic term extraction from small samples. __Proceedings of eLex 2025__.
